## System Architecture
This API follows a clean architecture pattern to ensure maintainability and scalability.
* **Controller Layer:** Handles HTTP requests and routing.
* **Service Layer:** Contains the core business logic.
* **Data Access Layer:** Manages communication with SQL Server using Entity Framework Core.

## Tech Stack
* **Framework:** ASP.NET Core 8.0 (or your version)
* **Language:** C#
* **Database:** Microsoft SQL Server
* **ORM:** Entity Framework Core
* **Tools:** Visual Studio 2022, SSMS, Postman
